To uninstall blackstone5 completely you must perform the following steps:

-Delete the "blakstone" folder found in /opt/lampp/htdocs.

-Delete the database, which you can find in the browser in the following path: "http://localhost/phpmyadmin/index.php?route=/database/structure&db=blackstone5"

-Delete the blackstone5 icon found in the "/usr/local/bin/blackstone5" directory
